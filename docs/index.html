AvaGotchi

M5Stack Basic / Gray で動く、たまごっち風ペット育成ゲーム

AvaGotchi ってなに？

AvaGotchi は、手のひらサイズのコンピューター「M5Stack」で動く、たまごっち風のペット育成ゲームです。 ごはんをあげたり、ミニゲームで遊んだり、お世話をしながらペットを育てましょう。 育てた子の個性はデバイスごとに異なり、QRコードで「じまんカード」を表示してシェアできます。

名前の 「Ava」 にはいくつかの意味が込められています：

  • Avatar：人の分身として、遠隔でも"自分"として振る舞える存在
  • Avant-garde：従来のロボット観を超えた、前衛的な存在
  • Available：場所や身体の制約を超えて、いつでもどこでも"利用可能"
  • Avail：人の可能性を最大限に活かし、社会参加に役立つ
  • Avaricious（再解釈）：より多くの体験・つながりを貪欲に求める人間性を象徴

必要なもの

  • M5Stack Basic または Gray（どちらでも動きます。Grayは傾きセンサーを使ったゲームも追加されます）
  • USB-C ケーブル（データ転送対応のもの。充電専用ケーブルは不可）
  • パソコンに Google Chrome または Microsoft Edge（スマホ・Safari・Firefoxは非対応）

M5Stack は スイッチサイエンス などで購入できます。 パソコンとブラウザがあれば、アプリのインストールは不要です。

① 書き込む

M5Stack Basic / Gray を USB でつないで、下のボタンを押すだけ。アプリのインストールは不要です（機種は自動判定されます）。

このブラウザは Web Serial に対応していません。
パソコンの Chrome または Edge で開いてください（スマホ・Safari・Firefox は非対応）。 シリアルポートへのアクセスが許可されませんでした。もう一度お試しください。

ボタンを押すとシリアルポート選択画面が出ます。「CP210x …（COMxx）」を選んで Connect → Install と進めてください。書き込みには1〜2分かかります。

⚠️ インストールは自己責任で行ってください。本ソフトウェアの書き込み・利用によって生じた いかなる損害（デバイスの不具合・データ消失等）についても、作者は一切の責任を負いません。

② 遊び方（操作）

A ごはんをあげる
B ミニゲーム選択
C 短押し：ステータス画面
C 長押し：天気取得＋時刻合わせ
🔄 くるくる回す → 目が回る ※
👆 トンとつつく → くすぐったい ※
🙃 逆さにする → びっくり ※
😴 ほうっておく → おやすみ

ミニゲームは B ボタンだけで遊べます（5びょうあて／反射神経／連打／ラッキー7／がまんくらべ）。 IMU 搭載の Gray では「かたむけて エサとり」も追加されます。 ごきげんと満腹を高く保つと、たまごを産んで次の世代が生まれます。
※ は IMU 搭載機（Gray）のみ。Basic 無印では動作しません。

③ うまくいかないとき

  • 「Failed to initialize」やリセットを繰り返す → 書き込みモードに入れていません。次を順に試してください：
    1. 他のシリアル接続をすべて閉じる（Arduino IDE・PlatformIOモニタ・M5Burner・別タブ）
    2. INSTALL を押し、「Connecting…」表示中に本体の赤い電源ボタンを1回カチッと押す（手動リセット）
    3. USB ケーブルを変える／PC に直挿し（充電専用ケーブル・ハブは不可）
    4. Chrome と Edge を切り替えてみる
  • ボタンが灰色のまま → スマホや非対応ブラウザです。PCの Chrome / Edge で開いてください。
  • ポート一覧に出てこない → CP210x USB ドライバ（Silicon Labs）が必要な場合があります。
  • 天気・時刻機能は WiFi 設定が必要です（配布版は未設定。ソースの secrets.h 参照）。

AvaGotchi（あばごっち）

M5Stack Basic / Gray で動く、たまごっち風ペット育成ゲーム

🤔 AvaGotchi（あばごっち）ってなに？

手のひらサイズのコンピューター「M5Stack」で動く、たまごっち風のペット育成ゲームです。 ごはんをあげたり、ミニゲームで遊んだり、お世話をしながら自分だけのペットを育てましょう。 ペットの見た目や性格はデバイスごとにちがっていて、QRコードで「じまんカード」を出して友だちに自慢したり、SNSでシェアしたりできます。

「Ava（アバ）」という名前について

「Avatar（アバター）」とは、ゲームやSNSで自分の代わりに使うキャラクターのこと。 あばごっちのペットも、あなたのお世話しだいで喜んだり悲しんだりする"あなただけの相棒"です。「Ava」にはそんな思いが込められています。

さらに「Ava」は、こんな言葉の頭文字でもあります：

  • Avant-garde（アヴァンギャルド）：これまでのゲームにはなかった、新しい形の仲間
  • Available（アベイラブル）：時間も場所も選ばず、いつでもそばにいてくれる
  • Avail（アベイル）：あなたの毎日にちょっとの楽しさと癒しをプラスする
  • Avaricious（アヴェアリシャス）：あなたとの体験やつながりを、もっともっとほしがっている
🎭 ペットの豊かな表情（喜び・悲しみ・眠り…）は、 M5Stack-Avatar ライブラリが描いています。スタックチャンコミュニティが育てたこのライブラリへのリスペクトも、「Ava」の名前に込められています。

📦 必要なもの

  • 🖥️ M5Stack Basic または Gray（どちらでも動きます。Grayは傾きセンサーを使ったゲームも追加されます）
  • 🔌 USB-C ケーブル（データ転送対応のもの。充電専用ケーブルは不可）
  • 💻 パソコンに Google Chrome または Microsoft Edge（スマホ・Safari・Firefox は非対応）

M5Stack は スイッチサイエンス などで購入できます。 パソコンとブラウザがあれば、アプリのインストールは不要です。

🥚 オプション：たまご型ケース
3Dプリンターがあれば、M5Stack にぴったりのたまご型ケースが作れます。より「たまごっち」らしい見た目になります。

① 書き込む

M5Stack を USB でつないで、下のボタンを押すだけ。アプリのインストールは不要です（機種は自動判定されます）。

このブラウザは Web Serial に対応していません。
パソコンの Chrome または Edge で開いてください（スマホ・Safari・Firefox は非対応）。 シリアルポートへのアクセスが許可されませんでした。もう一度お試しください。

ボタンを押すとシリアルポート選択画面が出ます。「CP210x …（COMxx）」を選んで Connect → Install と進めてください。書き込みには 1〜2 分かかります。

⚠️ インストールは自己責任で行ってください。本ソフトウェアの書き込み・利用によって生じたいかなる損害についても、作者は一切の責任を負いません。

🎮 ② 遊び方（操作）

A ごはんをあげる
B ミニゲーム選択
C 短押し：ステータス
C 長押し：天気・時刻
🔄 くるくる → 目が回る ※
👆 トンとつつく → くすぐったい ※
🙃 逆さにする → びっくり ※
😴 ほうっておく → おやすみ

ミニゲームは B ボタンだけで遊べます（5びょうあて／反射神経／連打／ラッキー7／がまんくらべ）。 IMU 搭載の Gray では「かたむけてエサとり」も追加されます。 ごきげんと満腹を高く保つと、たまごを産んで次の世代が生まれます。
※ IMU 搭載機（Gray）のみ。Basic 無印では動作しません。

📶 ③ WiFi を設定する（天気・時刻機能）

天気の取得と時刻合わせには、自宅のWiFiを設定する必要があります。 microSDカード（M5Stack に差し込んで使うカード）に設定ファイルを作るだけでOKです。

  1. microSDカードを開き、wifi.txt という名前のテキストファイルを作る
  2. 中身を以下の形式で保存する： 
    自宅のWiFi名
パスワード
  3. microSDカードを M5Stack に差し込んで電源を入れる
  4. 設定が読み込まれると wifi_done.txt に自動でリネームされます
  5. 以降は C ボタン長押し で天気と時刻を取得できます

⚠️ WiFi パスワードはデバイス本体に保存されます。設定後は wifi_done.txt をSDカードから削除することをおすすめします。

❓ ④ うまくいかないとき

  • 「Failed to initialize」やリセットを繰り返す → 書き込みモードに入れていません：
    1. 他のシリアル接続をすべて閉じる（Arduino IDE・PlatformIO・M5Burner・別タブ）
    2. INSTALL を押し、「Connecting…」中に本体の赤い電源ボタンを1回押す
    3. USB ケーブルを変える／PC に直挿し（充電専用ケーブル・ハブは不可）
    4. Chrome と Edge を切り替えてみる
  • ボタンが灰色のまま → スマホや非対応ブラウザです。PC の Chrome / Edge で開いてください。
  • ポート一覧に出てこない → CP210x USB ドライバ（Silicon Labs）が必要な場合があります。
  • 天気・時刻が動かない → 上の「③ WiFi を設定する」の手順でSDカードから設定してください。
